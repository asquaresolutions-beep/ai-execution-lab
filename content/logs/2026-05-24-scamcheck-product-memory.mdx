## Documented Failure Lineage

Three direct failures + two proactively avoided from TrustSeal experience:

**Failures hit and resolved:**
1. **CSS specificity conflict** (2026-01-28) — Color class conflicts between probability bar and verdict card. Resolved by inline styles for computed values, CSS custom properties for themed values.
2. **Gemini 429 rate limit — no UX** (2026-02-10) — [Full report](/failures/gemini-rate-limit-429-no-ux) — 2 hours to resolve — structured error response pattern established
3. **GA4 cookie_domain not scoped** (2026-02-05) — [Cross-domain tracking report](/failures/ga4-cross-domain-tracking-gap) — 30 min to resolve — cookie_domain set explicitly

**Proactively avoided (learned from TrustSeal):**
- Firebase Functions Node 22 — set explicitly from first deploy, never hit Node 18 crash
- Firebase Auth custom domain — added to Authorized Domains before first custom domain deploy
