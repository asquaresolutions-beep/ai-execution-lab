## H) Weekly optimization loop
Each week:
1. **Content priorities:** pull `/api/scam-intel/growth?view=leverage` (RPM × demand × momentum) → publish/promote the top movers.
2. **Highest-performing entities:** `?view=entities` (fastest-growing) → double down with combos + Hindi.
3. **Highest CTR opportunities:** GSC Performance → pages with high impressions + low CTR → improve title/headline (honest).
4. **Best internal-link opportunities:** link new trending pages from the hub + related type pages (keep depth ≤3, 0 orphans).
5. **Strongest backlink opportunities:** `?view=backlink` → pitch the widget to the matching audience (fintech/news/forums).
6. **Discover:** promote in-season hub + freshest tier-1 pages; verify large-image previews.

> Bottleneck rule: only add infrastructure if a real one appears (e.g. cost spike → lower caps; crawl stall → add internal links; throughput → split AI calls to a worker). Otherwise: publish, distribute, monitor, compound.
